# Oportunidad de accesibilidad de Preflight

![Oportunidad de accesibilidad de Preflight](./assets/accessibility/hero.png){align="center"}

La oportunidad de accesibilidad de las Preflight en Sites Optimizer le ayuda a identificar y abordar los problemas de accesibilidad de su sitio web para garantizar que todos los usuarios, incluidas las personas con discapacidades, puedan utilizarla. Esta oportunidad se centra en evaluar el cumplimiento de su sitio con los estándares de accesibilidad y proporciona recomendaciones procesables para la mejora.
