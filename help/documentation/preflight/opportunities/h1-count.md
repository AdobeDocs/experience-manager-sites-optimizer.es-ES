# Oportunidad de recuento H1 de Preflight

![Oportunidad de recuento H1 de Preflight](./assets/h1-count/hero.png){align="center"}

La oportunidad de recuento H1 de Preflight en Sites Optimizer le ayuda a identificar y abordar los problemas relacionados con el uso de encabezados H1 en el sitio web. Esta oportunidad se centra en evaluar la estructura del sitio y proporciona recomendaciones procesables para la mejora.
